## Task 2: Trigger response wrapping

Left sudo su

```bash
exit
```

Create a response wrapping to secret/wescale

```bash

vault secrets enable -path secret kv-v2

vault kv put secret/wescale company="wescale"

vault kv get -wrap-ttl=360 secret/wescale
```
